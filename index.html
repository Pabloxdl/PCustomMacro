Why PCustomMacro

Simple blocks. Serious depth.

Start with a single trigger and one action. Keep going until you've built something that would normally take a script — without writing one.

🧩

Block-based chains

Every macro is a chain of blocks you connect visually. Reorder, branch, or duplicate a chain in seconds.

⚙️

Advanced when you need it

Conditions, loops, variables, and delays sit right alongside the basics — build as simple or as deep as the task requires.

🗂️

Many functions, one library

A growing set of built-in blocks covers input, timing, window targeting, and more, all reusable across chains.

🔄

Always up to date

This page tracks GitHub Releases directly — the newest build is always one click away, no manual updates needed.

How it works

A chain reads left to right

A trigger starts it, logic blocks decide what happens next, and action blocks do the work. Here's a real shape a chain can take.

Trigger On hotkey
Logic If condition
Action Move mouse
Action Click + hold
Logic Wait / delay
